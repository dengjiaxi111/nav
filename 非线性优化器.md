# ROS2 + NMPC (CasADi + acados) 项目环境配置指南

本项目将使用 **CasADi 构建系统动力学模型**，并通过 **acados 生成实时 NMPC 控制代码**，最终在 ROS2 Humble + C++ Controller 节点中运行，目标 50Hz+ 控制频率。

本文件旨在帮助你从零搭建环境，并说明如何将建模脚本与 ROS2 workspace 整合。

---

## 1. 依赖环境

推荐环境：

- Ubuntu 22.04
- ROS2 Humble
- GCC >= 9.4
- Python 3.10
- CMake >= 3.16

> acados 会通过 Python 依赖生成求解器，然后编译成高性能 C 代码供 C++ 直接使用。

---

## 2. 安装 CasADi

CasADi 可直接通过 pip 安装（包含自动微分核心功能）

```
sudo apt update
pip install casadi==3.6.5
```

验证是否可用：

```
python3 - <<EOF
import casadi as ca
x = ca.SX.sym("x")
print("CasADi OK:", x)
EOF
```

---

## 3. 安装 acados（推荐源码编译）

### 克隆仓库

```
git clone https://github.com/acados/acados.git --recursive
cd acados
```

### 安装依赖

```
sudo apt install -y \
    cmake build-essential \
    libblas-dev liblapack-dev \
    libqpOASES-dev libeigen3-dev \
    python3-matplotlib python3-numpy python3-scipy python3-pip
pip install -r requirements.txt
```

### 编译

```
mkdir build && cd build
cmake .. -DACADOS_WITH_QPOASES=ON -DACADOS_WITH_OSQP=ON
make -j$(nproc)
sudo make install
```

完成后目录结构：

```
/usr/local/lib/libacados.so
/usr/local/include/acados
/usr/local/lib/cmake/acados
```

验证：

```
acados_template --help
```

若有输出即成功。

---

## 4. 项目目录建议结构

```
ros2_ws/
├── src/
│   ├── wheelleg_control/          ◀ 你的controller包 (C++)
│   ├── nmpc_solver/               ◀ acados生成代码目录
│   ├── model_ocp/                 ◀ CasADi 建模 + OCP导出脚本
│   │   ├── model.py
│   │   ├── export_ocp.py          ◀ 运行后生成 solver
│   │   └── requirements.txt
│   └── wheelleg_msgs/
└── install/
└── build/
```

作用说明：

| 文件夹 | 内容 |
|---|---|
| model_ocp/ | 你用 CasADi 定义动力学模型 + OCP |
| nmpc_solver/ | acados 自动生成的求解器源码，C/C++ 可直接调用 |
| wheelleg_control/ | ROS2 Controller，加载 solver，订阅 odom 输出 cmd_vel 或 acc |

---

## 5. 执行模型导出 → 生成 NMPC solver

```
cd ros2_ws/src/model_ocp
python3 export_ocp.py
```

运行成功后会生成：

```
../nmpc_solver/libacados_ocp_solver.so
../nmpc_solver/c_generated_code/
```

---

## 6. 在 C++ ROS2 Controller 中使用 solver

在 `CMakeLists.txt` 中链接：

```
target_link_libraries(wheelleg_control
    acados
    hpipm
    blasfeo
    # 加入 acados 导出的 solver 静态库
    ${CMAKE_SOURCE_DIR}/../nmpc_solver/libacados_ocp_solver.so
)
```

Controller 伪代码示例：

```cpp
#include "acados_solver_wheelleg.h"   // export_ocp.py 自动生成

acados_ocp_capsule *capsule = wheelleg_acados_create();

while(running){
    set_state(...)
    set_reference(...)
    wheelleg_acados_solve();
    read_control_output(...)
}
```
