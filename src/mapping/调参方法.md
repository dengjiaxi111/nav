# 调参方法
- *1核心几何约束(台阶高度)*
参数调整	影响	适用场景
height_tolerance 增大 (0.04 → 0.06)	✅ 更宽松，容易识别<br>❌ 误识别增加	台阶制作粗糙、磨损严重
height_tolerance 减小 (0.04 → 0.02)	✅ 精确匹配，误报少<br>❌ 识别率下降	标准台阶、高精度要求
single_stair_height 修改	必须匹配实际台阶规格	非标准台阶 (如 18cm)

- *2台阶形态*
参数	增大	减小
min_stair_width	❌ 可能漏检窄台阶<br>✅ 过滤柱子	✅ 检测更多候选<br>❌ 误识别柱子
min_stair_depth	❌ 漏检薄台阶<br>✅ 过滤墙壁边缘	✅ 检测边缘凸起<br>❌ 误报增加
max_z_thickness	✅ 允许厚台阶<br>❌ 误识别墙壁	✅ 严格过滤墙壁<br>❌ 漏检厚台阶

- *2ROI 范围设置（影响检测范围）*
参数	调整方向	影响	适用场景
roi_x_max	⬆️ 增大 (2.0 → 3.0)	✅ 提前检测<br>❌ 远处误报增加	高速接近台阶
⬇️ 减小 (2.0 → 1.5)	✅ 减少误报<br>❌ 反应时间短	慢速精确接近
roi_x_min	⬆️ 增大 (0.2 → 0.3)	✅ 避免盲区噪声<br>❌ 近距离无法检测	传感器盲区大
roi_y_min/max	⬆️ 扩大 (±0.6 → ±1.0)	✅ 斜向接近也能检测<br>❌ 侧面误报	机器人可能斜向接近
roi_z_min	⬆️ 增大 (0.05 → 0.10)	✅ 彻底切掉地面<br>❌ 漏检低台阶	地面噪声严重
roi_z_max	⬆️ 增大 (0.5 → 0.8)	✅ 检测高台阶<br>❌ 计算量增加	存在高台阶


- *3聚类参数（影响分割效果）*

参数	增大	减小
cluster_tolerance	✅ 容忍点云稀疏<br>❌ 可能合并多个物体	✅ 精细分割<br>❌ 可能过度分割
min_cluster_size	✅ 过滤小噪声团<br>❌ 漏检小台阶	✅ 检测小物体<br>❌ 噪声干扰
max_cluster_size	✅ 允许大台阶<br>❌ 内存占用增加	✅ 防止异常大簇<br>❌ 可能截断台阶

- *4时间跟踪参数（影响稳定性）*
参数	增大	减小
min_detection_frames	✅ 更稳定，减少误锁定<br>❌ 反应慢	✅ 快速响应<br>❌ 容易误触发
lowpass_alpha	✅ 快速跟踪新位置<br>❌ 位置抖动	✅ 平滑稳定<br>❌ 响应迟钝
blind_zone_distance	✅ 更早进入盲区<br>❌ 盲区时间长	✅ 延后进入盲区<br>❌ 近距离检测失效


- *5预筛选参数（影响性能）*
参数	调整	计算量	识别率
cell_size_xy	增大 (0.05 → 0.10)	⬇️ 减少 50%	⬇️ 精度下降
min_cell_height	增大 (0.10 → 0.15)	⬇️ 过滤更多	⬇️ 可能漏检
max_cell_height	减小 (0.45 → 0.35)	⬇️ 过滤墙壁	⬆️ 减少误报
